1. Open PowerShell as Administrator, and execute:
  2. Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Windows-Subsystem-Linux
  3. (New-Object Net.WebClient).DownloadFile("https://aka.ms/wslubuntu2004", "Ubuntu2004.appx")
  4. Start-Process -FilePath "Ubuntu2004.appx"
  5. Launch
  6. On error page, press any key to continue...
  7. create UNIX username: same as brach:akrygows
  8. password can be anything
  9. when finished: exit
  10. back to Powershell window: bash
  11. if all steps complete, then you've fully installed BASH on your local machine for git.

1. Open PowerShell as Administrator, and execute:
  2. dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart
  3. dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart
  4. restart machine
  5. http://10.88.48.21:5000/download?fname=wsl_update_x64.msi
  6. Install wsl.msi package
  7. Open PowerShell as Administrator, and execute:
  8. wsl --set-default-version 2
  9. bash
  10. git
  11. if all steps complete, then you've fully installed BASH on your local machine for git.
